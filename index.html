Computer Hardware Basics

This site explains important computer hardware parts in a simple way.

CPU

The brain of the computer that processes instructions.

RAM

Temporary memory used while programs are running.

Storage

Stores files permanently like HDD and SSD.

Motherboard

Main board connecting all components.

GPU

Handles graphics and visual output.

Power Supply

Provides electricity to the system.

Cooling Fan

Keeps the system from overheating.

Sound Card

Manages audio input and output.

Network Card

Allows connection to the internet.
